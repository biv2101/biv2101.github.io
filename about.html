My name is Blagoj Ivanov.
My passion for coding started when I was 7 years old. It was when I discovered Scratch, a block coding application that sparked my main passion in life.
The first time I actually wrote code was in 4th grade. It was in SmallBasic, a programming language made for learning.
I made a couple DOS programs and I started with GUIs.
In the meantime I played a lot of Minecraft on many different servers. That's when I thought that I should try making plugins and that's when I started learning my first high-end programming language: Java.
I made a couple of plugins for a Macedonian server. Sadly, I lost all of my code due to a disk failure, but my passion for Java hasn't ended, and I am more confident coding in Java than any other language.
In high-school I started C++ and I also tried competing in the programming competitions, only to get a third place in 2022.
Now I am a Software Engineering and Information Systems student at the Faculty of Computer Science and Engineering with a passion for Data Science, Artificial Intelligence and Machine Learning.
